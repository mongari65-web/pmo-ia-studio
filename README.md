# PMO-IA Studio v2.0

## Installation

```bash
npm install
cp .env.example .env.local
# Remplir .env.local avec vos clés
npm run dev
```

## Flux principal
1. Créer un projet → /projects/new (Wizard 5 étapes)
2. Ouvrir le projet → /projects/:id
3. Générer les documents PMI liés au projet
4. Consulter l'historique des documents
